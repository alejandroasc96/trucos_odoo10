# Generar una clave publica
```
ssh-keygen
```

# Como ver la llave
```
cat .ssh/id_rsa.pub
```

# Como veo la llave RUT
```
cat ~/.ssh/id_rsa.pub
```

# Como clonamos desede Bitbucket
```
git clone git@bitbucket.org:marlonodoo/odoo_general.git
```
